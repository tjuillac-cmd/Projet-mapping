# SAMREADER

 `samreader.py` is a Python tool that parses a SAM mapping file and produces: a per chromosome summary table in a text file, coverage distribution plots along each reference sequence colored by mean MAPQ. User is allowed to choose various analysis settings in order to best fit his needs.

 This work was carried out as part of a Master's degree project in Bioinformatics at the University of Montpellier.

## Usage

    python3 samreader.py path/to/file.sam
